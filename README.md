# pastebot

一个监控 pastebin 的敏感内容，并发微博的 bot

### 安装

使用 pip 安装:
```
$ pip install -U pastebot
```

或使用 GitHub 安装
```
$ pip install -U git+https://github.com/fate0/pastebot.git
```

或使用源码安装:

```
$ python setup.py install
```

### 依赖
